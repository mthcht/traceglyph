What is TraceGlyph?

TraceGlyph is a free, open-source browser extension that performs real-time security analysis on every website you visit. It acts as a passive security toolkit running in your browser - detecting fingerprinting attempts, analyzing network behavior, identifying phishing indicators, extracting IOCs, and mapping findings to the MITRE ATT&CK framework.

Unlike blockers or privacy tools, TraceGlyph does not modify page behavior. It silently observes and records - giving you a forensic-grade snapshot of what every website is actually doing behind the scenes.

Built by mthcht for SOC analysts, threat hunters, incident responders, penetration testers, and anyone who wants to understand what websites are doing to their browser.

Data Collected (urlscan.io-equivalent)

TraceGlyph captures the same categories of data as urlscan.io scans, but in real time from your browser:

  • Requests - all HTTP transactions with method, URL, type, IP, status code
  • Cookies - name, value, size, tracking classification
  • Console output - log/warn/error/debug messages from page scripts
  • Page links - all <a href> elements with text and external flag
  • JS globals - non-standard window properties with type information
  • Page timing - DNS, TLS, TTFB, DOM loaded, transfer size, protocol
  • Technologies - Wappalyzer-class detection across DOM, globals, headers, and URLs
  • Script hashes - SHA-256 of inline script content and external script URLs
  • Security headers - 12+ response headers audited for presence and strength
  • Redirect chains - full chain with IPs and status codes
  • Domain→IP map - every hostname resolved to its IP address

Self-Filtering

TraceGlyph excludes its own activity from analysis results. The extension's own script loading, API calls, and DOM modifications are filtered out at every level:

  • Injected hooks check the call stack - if the caller frame originates from chrome-extension://, the detection is silently dropped
  • Network listeners skip all chrome-extension:// and moz-extension:// URLs
  • DOM MutationObserver ignores script elements with extension-origin src
  • fetch/XHR hooks skip extension URLs from exfiltration tracking

  1. Open your extensions page: chrome://extensions (or edge://extensions for Edge)
  2. Click the puzzle icon in the top-right corner of your browser
  3. Find TraceGlyph by mthcht and click the pin icon
  4. The shield appears with a live badge showing detection count and threat color

Privacy

TraceGlyph processes everything locally in your browser. No data is ever transmitted to any external server. No analytics, no telemetry, no cloud processing. All analysis happens on-device using browser extension APIs (webRequest, webNavigation, content scripts) available in Chrome, Edge, and Brave. The extension requires broad host permissions (<all_urls>) solely to monitor network requests across all sites - this data never leaves your machine.
